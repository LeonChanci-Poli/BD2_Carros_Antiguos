# CARROS_ANTIGUOS
Proyecto de Carros Antiguos en Nodejs y MongoDB.

# Tener Instalado: 
1. MongoDB
2. Node.js
3. IDE Visual Studio Code
4. Programa para versionamiento en GIT (SourceTree, GitKraken, GitHub-Desktop, etc)
5. Opcional (MongoDB Compass - Interfaz de MongoDB)
